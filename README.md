## Run Code 


### 1. Create the Virtual Environment
```
python -m venv myenv
```
### 2.  Activate the Virtual Environment:

```
myenv\Scripts\activate

```

### 3. Install Packages in the Virtual Environment
```
pip install opencv-python tk ultralytics
```
### 4 Run Projects
```
python main.py
```
